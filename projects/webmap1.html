Coconut Palm Health Monitoring System

üìò Project Description

The Coconut Palm Health Monitoring System is an advanced geospatial application designed to monitor and assess the health of coconut palm trees in large estates using Drone imagery and data analysis. This system processes high-resolution Drone images to evaluate the vegetation condition of coconut palms by calculating vegetation indices, allowing agricultural managers and researchers to detect stressed, under-performing, or unhealthy trees early. The system visualizes tree health on an interactive map dashboard, enabling users to make data-driven decisions for timely intervention, optimal resource allocation, and improved crop management and yield.
